# Miningbox

Dark colorscheme based in Gruvbox (Fantastic!) colorscheme

## Installation

Using [vim-plug](https://github.com/junegunn/vim-plug)

```vim
Plug 'freddiegar/miningbox.vim'
```

Add this to your .vimrc

```vim
set termguicolors

colorscheme miningbox
```
> Vim: 8
